| 配置项  | 必填 | 默认值 |   说明   |
| :-----: | :--: | :----: | :------: |
| 配置项1 |  是  |   无   | 配置说明 |
| 配置项2 |  否  |   无   | 配置说明 |

### 指令表
| 指令  | 权限 | 需要@ | 范围 |   说明   |
